# GymTracker

Приложение для отслеживания тренировок в спортзале.

## Функционал

- ✅ Запись тренировок (упражнения, подходы, вес, повторения)
- ✅ Справочник упражнений с привязкой к группам мышц
- ✅ Статистика по группам мышц за неделю/месяц/год
- ✅ Графики прогресса по весам
- ✅ Импорт/экспорт данных в JSON
- ✅ Офлайн-режим (локальная SQLite база)

## Технологии

- Kotlin
- Jetpack Compose
- Room (SQLite)
- Vico (графики)
- Material Design 3

## Сборка

### Локально

```bash
./gradlew assembleDebug
```

APK будет в `app/build/outputs/apk/debug/`

### GitHub Actions

1. Сделайте push в репозиторий
2. APK будет в артефактах workflow
3. Для релиза создайте тег `v1.0.0` — APK появится в Releases

## Импорт данных из MD-файла

Используйте Python-скрипт `parser/md_to_json.py`:

```bash
python3 parser/md_to_json.py путь/к/файлу.md
```

Результат — `gym_data.json`, который можно импортировать в приложение.

## Структура базы данных

- **muscle_groups** — группы мышц (иерархия)
- **exercises** — упражнения с привязкой к группам
- **workouts** — тренировки (по датам)
- **workout_exercises** — упражнения в тренировке
- **sets** — подходы (вес, повторения)
